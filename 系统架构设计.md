## 前端

| 技术     | 选型         | 作用                                 |
| -------- | ------------ | ------------------------------------ |
| 前端框架 | Vue 3        | 构建系统管理界面，负责数据展示与交互 |
| 编程语言 | TypeScript   | 提升代码规范性和可维护性             |
| 构建工具 | Vite         | 快速构建与打包前端项目               |
| UI组件库 | Element Plus | 快速搭建后台管理界面                 |
| 通信方式 | Axios        | 与后端进行HTTP接口交互               |
| 状态管理 | Pina         |                                      |

## 后端

### 1. 后端框架选型

| 技术     | 选型        | 作用                       |
| -------- | ----------- | -------------------------- |
| Web框架  | FastAPI     | 提供高性能RESTful接口      |
| 编程语言 | Python      | 与深度学习模型无缝衔接     |
| 接口规范 | RESTful API | 前后端分离、接口统一       |
| 接口文档 | Swagger UI  | 自动生成接口文档，便于调试 |

### 2. 后端关键技术组件

| 技术       | 作用                       |
| ---------- | -------------------------- |
| JWT        | 实现用户身份认证与权限控制 |
| SQLAlchemy | 实现数据库对象关系映射     |
| Loguru     | 记录系统运行日志           |
| Uvicorn    | 高性能ASGI服务器           |
| Nginx      | 反向代理与负载均衡         |

## 数据库

| 类型         | 技术                         | 作用                                             |
| ------------ | ---------------------------- | ------------------------------------------------ |
| 关系型数据库 | MySQL                        | 存储用户信息与识别结果                           |
| 缓存数据库   | Redis(还可以用elasticsearch) | 缓存热点数据与状态信息                           |
| 文件存储     | 本地 / MinIO                 | 存储原始与增强图像                               |
| 远程存储桶   | 阿里云OSS                    | 存储用户上传的图像（可以用作后续模型的二次训练） |
