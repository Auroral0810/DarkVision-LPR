# 暗视 · 低光照车牌识别系统（DarkVision-LPR）

论文题目：基于YOLOv12与Retinex图像增强的低光照条件下车牌识别系统设计与优化

项目描述：低光照下的车牌识别（使用yolo12训练模型，OpenCV和Pytorch，用Retinex等技术实现图像增强）

| 技术         | 选型    | 作用                   |
| ------------ | ------- | ---------------------- |
| 深度学习框架 | PyTorch | 构建与加载YOLOv12模型  |
| 目标检测模型 | YOLOv12 | 实现车牌定位与检测     |
| 图像增强算法 | Retinex | 提升低光照图像质量     |
| 图像处理     | OpenCV  | 图像读取、裁剪与可视化 |

📌 **作用说明**
AI推理服务独立部署，负责低光照图像增强与车牌检测推理，减少业务逻辑与模型计算之间的耦合，提高系统整体稳定性与可扩展性。

本系统采用**前后端分离架构**，并将**深度学习推理服务独立部署**，整体架构具有良好的扩展性、可维护性和上线能力，满足后续向桌面端、移动端及小程序扩展的需求。

```text
前端展示层  →  后端业务层  →  AI推理服务层  →  数据存储层
```

## DarkVision-LPR 用户功能规划文档

### 一、普通用户（FREE）功能列表

#### 1.1 核心识别功能

- **单张图片识别**
  - 支持上传单张图片（JPG/PNG/BMP，最大5MB）
  - 实时显示识别进度条

  - 显示原图与增强后对比图

  - 查看最近7天的识别记录

  - 上传身份证正反面
  - 人脸识别验证
  - 认证后解锁更多功能（如批量识别体验）
- **使用统计**
  - 今日已用/剩余次数
  - 本月累计识别次数
  - 识别成功率统计图表（最近7天）

#### 1.3 其他功能

- **升级提示**
  - 用量不足时显示升级VIP引导
  - VIP功能对比表格
  - 一键跳转购买页面
- **帮助中心**
  - 快速入门教程（图文/视频）
  - 常见问题FAQ
  - 在线客服/智能机器人咨询
