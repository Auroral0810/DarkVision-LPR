# 前端架构

前端生态采用 **Vue 3** 全家桶进行构建，强调组件化、响应式与高性能。

## 技术选型

| 技术 | 选型 | 作用 |
| :--- | :--- | :--- |
| **核心框架** | Vue 3 (Composition API) | 构建响应式用户界面 |
| **构建工具** | Vite 5 | 极速冷启动与热更新 (HMR) |
| **语言** | TypeScript 5 | 静态类型检查，提升代码健壮性 |
| **UI 组件库** | Element Plus | 快速构建高质量 B 端/C 端界面 |
| **状态管理** | Pinia | 轻量级、直观的全局状态管理 |
| **路由管理** | Vue Router 4 | 单页应用 (SPA) 路由控制 |
| **HTTP 客户端** | Axios | 统一封装请求拦截与响应处理 |
| **图表库** | ECharts 5 | 复杂数据可视化（如 KPI 看板） |
| **CSS 预处理** | SCSS | 模块化样式编写 |

## 项目结构

前端代码库采用 Monorepo 风格管理（逻辑上分离，物理上在同一仓库）：

```
frontend/
├── official-website/   # 官方宣传网站 (SEO 优化，展示为主)
├── user-portal/        # 用户操作台 (识别、充值、历史记录)
└── admin-portal/       # 管理后台 (权限、监控、配置)
```

## 核心特性

- **国际化 (i18n)**: 内置多语言支持（中/英）。
- **主题定制**: 支持深色模式与品牌色配置。
- **权限指令**: 封装 `v-permission` 指令实现细粒度按钮级权限控制。
- **动态路由**: 基于后端返回的用户权限动态生成前端菜单与路由表。
