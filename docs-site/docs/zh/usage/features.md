# 功能介绍

DarkVision-LPR 为不同角色的用户提供了差异化的功能体验。

## 普通用户 (FREE)

适合个人开发者或轻量级体验。
- **单张识别**: 每日 20 次免费额度，支持 JPG/PNG 格式。
- **历史记录**: 保留最近 7 天的识别记录与图片。
- **基础统计**: 查看个人的每日使用情况。

## VIP 用户 (订阅制)

适合专业测试人员或小规模商用。
- **批量识别**: 支持一次性上传 50 张图片，并行处理。
- **视频识别**: 上传 MP4 视频，系统自动抽帧识别并生成检测报告。
- **高精度模式**: 解锁更大参数量的模型，提升 10% 识别率。
- **数据导出**: 支持将历史记录导出为 Excel/CSV。
- **永久存储**: 历史记录无时间限制，永久保存。

## 企业用户 (Enterprise)

适合大规模部署与团队协作。
- **无限额度**: 无限制的识别次数与并发数。
- **API 接入**: 获取 API Key，将识别能力集成到自己的业务系统中。
- **子账户管理**: 创建最多 50 个子账户，分配部门额度。
- **专属客服**: 7x24 小时技术支持与 SLA 保障。
- **私有化部署**: 支持将系统完整部署在企业内网环境。

## 管理员功能

- **用户管理**: 封禁用户、调整用户等级、实名认证审核。
- **系统监控**: 实时查看 CPU/GPU 负载、QPS、错误率。
- **内容管理**: 编辑官网公告、FAQ、轮播图。
- **模型热更**: 在线切换推理模型版本，无需停机。
