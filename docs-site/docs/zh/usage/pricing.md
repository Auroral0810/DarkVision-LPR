## 套餐详情

### 1. 社区版 (Community)
适合个人学习、毕业设计或简单的验证性测试。
- **完全开源**: 您可以获取基础版源码。
- **开箱即用**: 简单的 Docker 部署。
- **限制**: 不包含 VIP 专属的批量识别和视频分析功能。

### 2. 专业版 (Pro)
适合对识别精度和并发有一定要求的商业场景，如小区门禁、小型停车场。
- **高精度模型**: 提供更大参数量的 YOLO 模型，提升 10% 以上的准确率。
- **优先队列**: 您的 API 请求将获得更高的优先级。
- **增值功能**: 解锁批量识别、视频流分析、Excel 导出等高级功能。

### 3. 企业版 (Enterprise)
为大规模安防、智慧城市项目量身定制。
- **私有化部署**: 我们提供完整的部署脚本和技术指导，将系统部署在您的内网服务器上，数据绝对安全。
- **源码交付**: 提供完整的 Pro 版源代码（含加密算法模块）。
- **定制开发**: 每年赠送 5 人天的定制开发服务（如对接特定的摄像头协议）。
