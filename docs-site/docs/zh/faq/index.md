# 常见问题 (FAQ)

## 识别准确率相关

**Q: 为什么我的图片识别不出来？**
A: 请检查图片是否过于模糊，或车牌是否存在严重遮挡。虽然系统配备了 Retinex 增强，但物理遮挡无法通过算法恢复。此外，请确保图片分辨率至少在 720p 以上。

**Q: 支持哪些类型的车牌？**
A: 目前支持中国大陆常见的蓝牌、黄牌（单/双层）、绿牌（新能源）、白牌（警车）以及黑牌。

## 部署于运维

**Q: 必须要使用 GPU 吗？**
A: 不强制。系统支持 CPU 推理，但速度较慢（约 200-500ms/张）。推荐使用 NVIDIA GPU（如 T4, 3060 等）以获得毫秒级的实时体验。

**Q: 如何重置管理员密码？**
A: 可以运行后端提供的脚本 `python scripts/reset_admin.py` 进行重置。

## 商务与授权

**Q: 个人开发者可以免费商用吗？**
A: DarkVision-LPR 基础版遵循 MIT 协议开源，您可以免费商用。但 VIP 组件和云服务部分需要订阅授权。

**Q: 如何购买企业版？**
A: 请联系我们的商务团队 `business@darkvision.com` 或通过官网的“联系我们”页面提交需求。
