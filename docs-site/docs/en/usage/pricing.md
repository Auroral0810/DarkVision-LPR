## Plan Details

### 1. Community Edition
Suitable for individual learning, graduation projects, or simple validation tests.
- **Open Source**: Get the basic source code.
- **Out of the Box**: Simple Docker deployment.
- **Limitations**: Does not include VIP-exclusive batch recognition and video analysis features.

### 2. Pro Edition
Suitable for commercial scenarios requiring certain recognition accuracy and concurrency, such as community access control and small parking lots.
- **High Precision Model**: Provides YOLO models with larger parameters, improving accuracy by over 10%.
- **Priority Queue**: Your API requests will receive higher priority.
- **Value-added Features**: Unlock batch recognition, video stream analysis, Excel export, and other advanced features.

### 3. Enterprise Edition
Tailored for large-scale security and smart city projects.
- **Private Deployment**: We provide complete deployment scripts and technical guidance to deploy the system on your intranet servers, ensuring absolute data security.
- **Source Code Delivery**: Provide full Pro version source code (including encryption algorithm modules).
- **Custom Development**: 5 person-days of custom development service included per year (e.g., interfacing with specific camera protocols).
