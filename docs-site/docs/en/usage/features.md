# Features

DarkVision-LPR provides a wealth of functional modules, covering the entire process from recognition to management.

## 1. Real-time Recognition (Live Demo)
Accessible via the **User Portal**.
- **File Upload**: Supports dragging and dropping local images/videos for recognition.
- **Snapshot Capture**: Supports calling the device camera for real-time capturing.
- **Result Display**: Instantly returns the recognized license plate number, vehicle type, confidence, and visualized bounding boxes.
- **Enhancement Contrast**: Click the "Compare" button to view the effect before and after low-light enhancement.

## 2. History Records
All recognition records are permanently saved.
- **Search & Filter**: Query by license plate number, time range.
- **Export**: Supports exporting query results to CSV/Excel files.
- **Details**: Click on a record to view the raw image, enhanced image, and full JSON meta-information.

## 3. Data Dashboard (Analysis)
A large visual screen providing decision support for administrators.
- **Traffic Trend**: Hourly/Daily vehicle traffic line chart.
- **Vehicle Attribute Distribution**: License plate color distribution (Blue, Green, Yellow), Vehicle type distribution.
- **Top Plates**: List of frequently appearing vehicles (e.g., suspicious vehicles).

## 4. User Management (Admin)
- **User List**: Add, delete, and modify system users.
- **Role Assignment**: Assign Admin, VIP, or User roles.
- **Quota Management**: Adjust the daily API call limit for specific users.

## 5. System Settings
- **Profile**: Modify avatar, password.
- **Real-name Verification**: Upload ID card for automated real-name verification (connected to Public Security/Aliyun API).
