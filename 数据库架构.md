## 一、用户与认证相关表

| 表名                        | 说明                           |
| ------------------------- | ---------------------------- |
| `users`                   | 系统主用户表，存储登录账号、用户类型、状态及基础信息   |
| `user_profiles`           | 用户扩展信息表，用于实名信息、性别、生日等补充资料    |
| `user_memberships`        | 用户会员状态表，记录会员类型、起止时间及有效状态     |
| `sub_accounts`            | 企业子账户关联表，定义企业主账户与子账户及权限      |
| `real_name_verifications` | 实名认证表，存储身份证信息及审核状态           |
| `login_logs`              | 用户登录日志表，记录登录行为与结果            |
| `password_reset_tokens`   | 密码重置令牌表，用于找回密码安全校验           |
| `third_party_logins`      | 第三方登录绑定表（微信 / QQ / GitHub 等） |

| `video_recognition_frames` | 视频识别帧表，记录视频中每一帧的识别结果      |
| `recognition_models`       | 识别模型版本管理表                 |
| `recognition_presets`      | 用户自定义图像增强预设表（VIP 功能）      |
| `recognition_statistics`   | 用户识别统计表，用于限额与计费控制         |
